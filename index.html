Wedding RSVP/Information Website

This site was built as a resource for friends and family to easily find information about my wife and I's upcoming wedding celebration. It includes all the information necessary to attend the celebration, and has links to wedding registries and an RSVP submission form.

The site was built using Angular and uses firebase as a datastore for the RSVP data. Local Storage is used to prevent double-RSVPing. Data can easily be exported via the firebase console, or by downloading an XML file.
